# QGenesis Backend

This folder holds backend services for QGenesis.

- **`material_extraction/`** — Python FastAPI service for extracting text from PDFs, DOCX, PPTX, images, and other uploads. See [material_extraction/README.md](material_extraction/README.md) for setup and API details.

**Next step:** Firebase Firestore (and related server-side logic) will be integrated here.
